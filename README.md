# 🧠 AI Resume Analyzer

🚀 An AI-based project that compares a resume with a job description and calculates a match score using NLP techniques.

---

## ✨ Features
- 📄 Resume vs Job Description comparison  
- 📊 Match score using TF-IDF  
- 🤖 Basic AI (NLP) implementation  
- 💡 Suggests improvements  

---

## 🛠️ Tech Stack
- Python  
- NLTK  
- Scikit-learn  
- NLP  

---

## 📂 Project Structure
resume-analyzer/
│── app.py  
│── requirements.txt  
│── sample_resume.txt  
│── job_description.txt  

---

## ▶️ How to Run

1️⃣ Install dependencies  
pip install -r requirements.txt  

2️⃣ Run the project  
python app.py
